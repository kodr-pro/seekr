# Release Notes - v0.2.0: Security & Performance Overhaul 🛡️⚡

We are excited to announce the release of **Seekr v0.2.0**, a major update focused on hardening the core infrastructure and delivering a significantly more responsive experience. This "Security & Performance Overhaul" represents a complete redesign of our internal event systems and the introduction of advanced diagnostic tools.

## 🛡️ Security & Stability

### Seekr Doctor (`seekr doctor`)
We've introduced a built-in diagnostic suite to ensure your environment is safe and correctly configured.
- **Config Validation**: Audits your `config.toml` for syntax errors and missing provider keys.
- **Connectivity Checks**: Verifies secure handshake with AI providers (OpenAI, DeepSeek, Anthropic) before starting a session.
- **Permission Auditing**: Transparently checks and reports on working directory read/write permissions to prevent agent failures.

### Hardened Provider Management
- **Enhanced API Key Parsing**: Improved handling of environment variables and configuration keys to prevent accidental leaks or misconfigurations.
- **Validated Tool Approval**: Refined the manual tool approval flow to ensure absolute user control over agent actions.

---

## ⚡ Performance Improvements

### Overhauled Event Handling System
The backend event loop has been rewritten to prioritize UI responsiveness.
- **Minimized Latency**: Transitioned to a more efficient event dispatching model, resulting in snappier transitions between Focus modes (Input, Chat, Tasks).
- **Reduced CPU Overhead**: Optimized the TUI refresh cycle to only redetermine layouts when necessary.

### Optimized Chat Rendering
Experience smoother scrolling and faster rendering in long conversations.
- **Simplified Scroller**: Refactored the chat scrolling logic to eliminate the performance bottlenecks of the previous text-selection system.
- **Syntax Highlighting (Syntect)**: Integrated `syntect` for highly efficient, multi-threaded syntax highlighting in code blocks.

### Agent Efficiency
- **Memory Management**: Enhanced the way Seekr manages conversation history, reducing token overhead while maintaining high context relevance.
- **Improved Planning**: Refined the agent's internal reasoning loop, leading to faster tool execution and more concise responses.

---

## ✨ New Features & UX Enhancements

- **Unified Menu (Ctrl+G)**: A brand new global menu for managing Sessions, Models, Providers, and Settings in one place.
- **Seamless Setup Flow**: A completely redesigned onboarding process that guides you through provider configuration and API validation.
- **Live Version Alerts**: Seekr now notifies you directly in the TUI when a new version is available on GitHub.
- **Automatic Task Management**: Smarter task tracking that updates in real-time as the agent makes progress.

## 🐛 Bug Fixes
- Fixed a bug where terminal resizing would occasionally break the layout.
- Resolved an issue with deepseek-reasoner model selection.
- Improved error messaging when a tool execution fails.
