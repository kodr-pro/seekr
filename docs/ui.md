# Terminal UI

**seekr** features a rich, responsive Terminal UI (TUI) that organizes multiple streams of information into a cohesive interface.

## Layout Overview

The UI is divided into several persistent and contextual areas:

### 1. Navigation Tabs
At the top of the screen, you'll find the main navigation:
- **Chat:** Your conversation history with the AI.
- **Tasks:** Active goals and sub-tasks.
- **Setup:** Configuration management.

### 2. Interaction Area
The largest portion of the screen, where chat messages, reasoning, and tool executions are displayed.

### 3. Status Bar
Located at the bottom, showing the current state of the agent (Idle, Thinking, Running Tool), the selected model, and shortcut keys.

## Key Shortcuts

| Key | Action |
| :--- | :--- |
| `Tab` | Cycle through navigation tabs. |
| `Enter` | Send message / Confirm selection. |
| `Esc` | Cancel / Go back. |
| `Ctrl+C` | Force quit the application. |

## Themes
Currently, **seekr** supports a high-contrast dark theme optimized for most modern terminal emulators.
